# 校园帮

## 项目简介
校园帮是基于Java的校园互助平台。平台支持两种角色登录：普通用户和管理员。 普通用户可以发布悬赏任务，其他用户可以看到发布的悬赏任务，并接受任务完成。管理员可以对用户、任务、任务所属学校进行管理。适合作为大作业、课程设计、毕业设计。
